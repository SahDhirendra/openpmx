# OpenPMX — Open-Source Predictive Maintenance Platform

An open-source predictive maintenance platform for small and mid-sized 
manufacturers. Built to democratize industrial AI for the 300,000+ SMB 
manufacturers in the US who can't afford enterprise solutions.

## What it does
- Detects machine anomalies before they become failures
- Predicts remaining useful life (RUL) of industrial equipment
- Generates maintenance alerts and recommendations
- Runs fully on-premise — no cloud dependency, no vendor lock-in

## Tech Stack
- Python · FastAPI · TimescaleDB · React · Docker
- ML: LSTM · Isolation Forest · Scikit-learn
