# Data

This folder contains the datasets used by OpenPMX.

## NASA IMS Bearing Dataset

This project uses the NASA IMS Bearing Dataset collected by the 
University of Cincinnati's Center for Intelligent Maintenance Systems.

### How to download

1. Go to https://www.kaggle.com/datasets/vinayak123tyagi/bearing-dataset
2. Download and unzip the file
3. Place the folders (1st_test, 2nd_test, 3rd_test) inside this data/ folder

### Dataset structure
data/
├── 1st_test/     # 2156 snapshots - Oct to Nov 2003
├── 2nd_test/     # 984 snapshots - Feb 2004
└── 3rd_test/     # 4448 snapshots - Mar to Apr 2004
